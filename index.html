Codecademy CheatSheet - CSS Animation

@keyframes

The @keyframes rule specifies the animation code.

The animation is created by gradually changing from one set of CSS styles to another.

During the animation, you can change the set of CSS styles many times.

Specify when the style change will happen in percent, or with the keywords "from" and "to", which is the same as 0% and 100%. 0% is the beginning of the animation, 100% is when the animation is complete.

Syntax

          
    @keyframes animation_name {
    from (or 0%) {css-style} 
    XX% {css-style} (additionals % optional) 
    to (or 100%) {css-style} }

Example

          
    @keyframes code {
        from {background-color: black;}

        50% {color: hotpink;}

        to {background-color: navy;
            color: hotpink;}
    }

Animatable Properties

List of animatable properties from w3schools

Animation Properties

        
    animation: name duration timing-function delay iteration-count direction fill-mode play-state;

animation-duration

The animation-duration property defines how long an animation should take to complete one cycle.

Duration Animation
1s
2s

animation-timing-function

The animation-timing-function specifies the speed curve of an animation.

Timing function Animation
linear
ease
ease-in
ease-out
ease-in-out

animation-delay

The animation-delay property specifies a delay for the start of an animation.

Delay Animation
0s
0.5s

animation-iteration-count

The animation-iteration-count property specifies the number of times an animation should be played. Note that all animation on this page use the infinite value.

animation-direction

The animation-direction property defines whether an animation should be played forwards, backwards or in alternate cycles.

Timing function Animation
normal
reverse
alternate
